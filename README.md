## Key Points

### Purpose

Kraken Forge is a performance‑focused engineering hackathon aimed at
building real, production‑grade tools for the Kraken ecosystem.
Participants develop high‑performance SDKs, visualizations, or low‑code
strategy builders inspired by real engineering challenges inside Kraken.

### Overall Goals

-   Build reusable, open‑source components powered by Kraken's API.
-   Strengthen community contributions to crypto trading infrastructure.
-   Identify top engineering talent for potential fast‑track hiring.

### Important Dates

-   **Registration:** Nov 21 -- Dec 4
-   **Invitations:** Dec 5
-   **Hackathon:** Dec 8 -- 24
-   **Submission Deadline:** Dec 24
-   **Winners Announced:** Jan 2026

## Track 3 (Main Focus): Strategy Builder

### Overview

Track 3 focuses on building a **Strategy Builder**
A visual or programmatic interface enabling traders to design, simulate, and execute trading strategies using Kraken's API.

### Expected Outcome

A production‑ready tool that allows users to: - Compose strategies using
drag‑and‑drop or modular logic blocks
- Simulate strategies using real‑time or historical data
- Deploy strategies directly via Kraken's trading API
- Monitor performance, logs, and execution flow

The final product should emphasize: - UX clarity
- Low latency
- High reliability
- Clear separation of concerns
- Extensibility for future feature additions
